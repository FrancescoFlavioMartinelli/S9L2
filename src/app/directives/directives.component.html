Le directives servono ad applciare logica su precisi elementi html del template
Le directives possono essere create custom
Oggi vedremo le directives [ngClass], [ngStyle], *ngFor, *ngIf

per inserire delle classi a un elemento usiamo
[ngClass]="'classe1 classe2'" !!Attenzione che come valore va assegnata una stringa
[ngClass]="['classe1', 'classe2']"

Posso impostare il valore stringa della classe anche con una propretà

Posso anche impostare un array di stringhe per assegnare tutte le classi

Posso inserire un oggeto con delle stringhe come propreità e un boolean per indicare se la classe va inserita o meno

Posso usare le proprietà per decidere se una classe vada o meno assegnata

Posso indicare una proprietà css nello nel ngStyle impostando anche propreità come valori

Posso usare anche l'intero oggetto

Possiamo decidere di visualizzare o nascondere un elemento dal tmeplate in base a un controllo codizionale usando
*ngIf="condizione"

Possiamo anche indicare una parte di html da visualizzare come ELSE
*ngIf="condzione;else elseBlock"

nell'elemento else indicare
#elseBlock

Questo viene visualizzato solo se il testo sopra non è visibile

Possiamo far generare alla directive più copie di un blocco di codice con
*ngFor="let e of arr; index as i;first ad firstItem"


*ngFor="let e of arr" -> genera il blocco per opngi elemento, entro quel tag possiamo usare e come elemento dell'attuale giro del ciclo
index as i -> possiamo avere l'indice del giro nella variabile i
first as firstItem -> possiamo usare firstItem come controllo if="firstItem" allora siamo al primo elemento
